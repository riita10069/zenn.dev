# 表現の崩壊 (collapse) 防止

## ひとことで言うと
「2つの見え方を特徴空間で近づける」だけの自己教師あり学習をすると、モデルは中身を理解する代わりに「いつも同じ答えを出せば差はゼロ」というズルを覚えてしまうことがあります。これを表現の崩壊(representation collapse)と呼びます。ターゲットを固定する、予測ヘッドを片側だけ付ける、分散を保つ罰則を足す、幾何変換で予測を入力に縛り付ける、といった工夫でこのズルを封じます。

## 直感的な理解

学校の答え合わせで、先生が「2人の答えが一致していれば正解」とだけ言ったとします。すると賢い生徒2人は、問題を解く代わりに「2人とも常に『42』と書こう」と示し合わせれば、どんな問題でも必ず一致して満点になります。先生のルールは満たされるけれど、2人は何も学んでいません。

自己教師あり学習の「2つの見え方を近づけよ」という目標には、これとそっくりの抜け穴があります。同じ画像の少し違う加工どうしを特徴空間で似た値にせよ、あるいは現在から予測した将来特徴を実際の将来特徴に近づけよ、という課題で、両方を学習で自由に動かせるなら、最も簡単な達成法は「どんな入力でも同じ一定値を出す」ことなのです。たとえば両方が常にゼロベクトルを出せば差(MSE)は完璧にゼロ。損失だけ見れば大成功ですが、入力が変わっても出力が変わらないので、特徴は完全に無意味です。信号が赤でも青でも、前方に車がいてもいなくても同じ特徴を出すなら、その特徴は何の判断にも使えません。これが表現の崩壊、あるいは「自明解(trivial solution、考えなくても損失が下がる手抜きの解)に落ちる」と呼ばれる現象です。

崩壊が厄介なのは、損失の数値だけ見ると「順調に下がって大成功」に見えることです。損失曲線はきれいに底へ向かうのに、できあがった特徴は使い物にならない。だから崩壊は「損失以外の指標で監視しないと気づけない静かな失敗」なのです。

## 基礎: 前提となる概念

- 自己教師あり学習(SSL): 人手ラベルなしでデータ自身から学ぶ枠組み。ここで扱うのは「同じ対象の2つの見え方を特徴空間で一致させる」joint-embedding 系の手法です。2つの見え方は、同じ画像への別々のデータ拡張(切り抜き・色変え)や、現在と将来のフレームなどで作ります。
- 埋め込み / 特徴(embedding / feature): 入力を要約した数値ベクトル。良い埋め込みは「似た入力は近く、違う入力は遠く」配置されます。崩壊するとこの配置が1点に潰れます。
- 対比学習(contrastive learning): 「似ているペア(positive)は近づけ、似ていないペア(negative)は引き離す」学習。negative を引き離す力が「全部を1点に潰す」のを防ぎます。ただし大量の negative サンプルが必要で重いのが難点でした。
- stop-gradient(勾配停止): ある出力に対し、そこへ学習の修正信号(勾配)を流さない操作。実装上は `.detach()` や `requires_grad=False` で行います。
- EMA (Exponential Moving Average、指数移動平均): 過去の値を強く残しつつ新しい値を少しずつ混ぜる平滑化。`new = τ·old + (1-τ)·current`。
- 分散(variance): 出力がどれだけバラついているか。崩壊は「分散がゼロに潰れる」こととほぼ同義なので、分散を監視すると崩壊を早期検知できます。
- 完全崩壊と次元崩壊: 全次元が1点へ潰れるのが完全崩壊(complete collapse)、一部の次元だけが潰れて実効的な情報次元が減るのが次元崩壊(dimensional collapse)。後者は損失も分散の平均も正常に見えるため、さらに発見しづらいです。

## 仕組みを詳しく

崩壊を防ぐ中心原理は「予測側とターゲット側を非対称にする(asymmetry)」か「分散を直接保つ罰則を課す」かのどちらかです。両方が自由に動けるから両方ゼロに逃げられるので、片方を固定したり構造的に区別したり、潰れること自体を禁じたりして逃げ道を塞ぎます。

### 1. ターゲットを固定する(frozen / stop-gradient)

最も直接的な方法です。ターゲットを作る側のエンコーダを学習で動かさないようにします。

- frozen(凍結): ターゲット用エンコーダの重みを学習中ずっと固定する。
- stop-gradient: ターゲット側の出力に `.detach()` を掛け、勾配を流さない。

なぜ防げるか。ターゲットが固定されていれば、予測側は「動かない正解」に合わせるしかありません。正解が入力ごとに違う値(意味のある特徴)であれば、予測側も入力に応じて違う値を出さざるを得ず、「常に一定値」というズルが通用しなくなります。逃げ道(両方ゼロ)が片側固定で閉じるわけです。

```
ターゲット器(固定):  赤信号 → [0.9, 0.1]   青信号 → [0.1, 0.9]
予測側が常に [0.5,0.5] を出すと、どちらでも誤差が残る
→ 誤差を減らすには信号の色で出力を変えるしかない → 意味のある特徴を学ぶ
```

### 2. EMA でターゲットをゆっくり更新する(BYOL 方式)

完全固定の代わりに、ターゲットを学習側(online)の重みで EMA 追従させます。

```
for p_target, p_online in zip(target.parameters(), online.parameters()):
    p_target = τ * p_target + (1 - τ) * p_online    # τ ≈ 0.99〜0.999
```

`τ` が1に近いほどターゲットの変化は遅くなります。「ほぼ固定だが、学習が進んだぶんだけ少し追いつく」状態を作り、固定の安定性とターゲットも徐々に良くなる柔軟さを両立します。BYOL はこれに加えて online 側だけに predictor(予測ヘッド)を付ける非対称構造を持ち、negative なしで崩壊を防ぎます。ただしターゲットが動くぶん非定常になり、`τ` の設定が要になります(小さすぎると崩壊、大きすぎるとターゲットが更新されず学習が遅い)。

### 3. 分散・共分散の正則化(VICReg 方式)

非対称性に頼らず、崩壊そのものを罰則で直接禁止する別系統です。VICReg は損失に3つの項を足します。variance(各次元の分散を一定の閾値以上に保つ。潰れたら罰)、invariance(2つの見え方を近づける本来の目標)、covariance(次元間の相関を下げて情報が重複しないようにし、次元崩壊も防ぐ)。「出力が潰れる=分散ゼロ」なので、分散を下げさせない項が崩壊を構造的に封じます。stop-gradient も EMA も使わず両ブランチを対称に学習できるのが利点です。

### 4. 幾何ワープによるアンカー効果

将来予測を `Warp_ego(z_t) + Δ_dynamic` と分解する設計(別項参照)には、崩壊抑止の副次効果があります。`Warp_ego(z_t)`(現在特徴を既知の自車運動で未来視点へ座標変換したもの)が予測の大部分を最初から確定させるため、予測は自然と入力(現在特徴)の中身を反映した、入力ごとに異なる値になります。すべてを一定値に潰そうとしても warp 項が入力依存の構造を強制するので、自明解へ落ちにくい。ワープが予測を現実の入力に繋ぎ止める錨(アンカー)として働くわけです。これは frozen ターゲット(逃げ道を塞ぐ)とは別経路の防御で、両者は併用できます。問題固有の構造的事前知識(ここでは幾何)を予測に注入すると崩壊耐性が上がる、という一般原理の一例です。

## 手法の系譜と主要論文

- Chen et al., "SimCLR" (ICML 2020, arXiv:2002.05709) / He et al., "MoCo" (CVPR 2020, arXiv:1911.05722)。対比学習の代表。negative ペアを引き離す力で崩壊を防ぎますが、大量の negative(SimCLR は大バッチ、MoCo は巨大メモリバンク)が必要でした。「崩壊をどう防ぐか」という問いの出発点です。
- Grill et al., "Bootstrap Your Own Latent (BYOL)" (NeurIPS 2020, arXiv:2006.07733)。negative ペアなしでも崩壊しない自己教師ありを初めて示しました。online と EMA target の非対称構造 + predictor で自明解を回避。当時「なぜ崩壊しないか」の理論が不透明で、後続研究の議論を呼びました。
- Chen & He, "SimSiam" (CVPR 2021, arXiv:2011.10566)。EMA すら無くても「stop-gradient + predictor」だけで崩壊を防げることを実験で示しました。崩壊防止の本質が「非対称性(片側に勾配を流さないこと)」にあると明らかにした点が重要です。
- Bardes et al., "VICReg" (ICLR 2022, arXiv:2105.04906)。分散・不変・共分散の正則化で崩壊を直接禁止する別アプローチ。stop-gradient や EMA に頼らず対称な2ブランチで学習できる柔軟さを示しました。
- Jing et al., "Understanding Dimensional Collapse in Contrastive Self-Supervised Learning" (ICLR 2022, arXiv:2110.09348)。完全崩壊しなくても一部次元が潰れる「次元崩壊」を解析し、強い拡張や暗黙の正則化が原因になりうることを示しました。崩壊が二値ではなく程度問題であることを定量化した点で重要です。
- Assran et al., "I-JEPA" (CVPR 2023, arXiv:2301.08243)。画像で JEPA を実装。画素ではなく特徴空間でマスク領域を予測し、ターゲットエンコーダは context エンコーダの EMA、stop-gradient で崩壊を防ぎます。LeCun の JEPA 構想を実装に落とし、SSL の主要路線に位置づけました。

系譜を俯瞰すると、「negative で引き離す(SimCLR/MoCo)」→「非対称構造で negative を不要にする(BYOL)」→「非対称性の最小要素は stop-gradient だと突き止める(SimSiam)」→「非対称性に頼らず分散罰則で防ぐ(VICReg)」→「予測アーキテクチャ(JEPA)に統合する(I-JEPA)」という、崩壊防止の理解が段階的に深まる流れが見えます。

## 論文の実験結果(定量データ)

評価は主に ImageNet の線形評価(linear probing、凍結した特徴の上に線形分類器だけ載せて学習し、Top-1 正解率を測る。特徴の質を測る標準プロトコル)で行われます。

- BYOL (Grill et al., 2020): ResNet-50 バックボーンで ImageNet linear probe Top-1 約 74.3% を達成し、当時の SimCLR(約 69.3%)を明確に上回りました。negative なしで対比学習を超えたことが衝撃を与えました。アブレーションで、target を online と即同期(τ=0)にすると性能が崩れ、predictor を外すと崩壊する、と報告され、非対称構造が崩壊防止の鍵だと示されました。
- SimSiam (Chen & He, 2021): stop-gradient を外すと損失は一瞬で最小付近へ落ちるが、特徴の出力標準偏差がゼロへ潰れ(崩壊)、linear probe 精度がチャンスレベルへ落ちることを直接観測しました。stop-gradient を入れると ImageNet Top-1 約 71% 前後を達成。崩壊の有無を「出力の標準偏差」で可視化したアブレーションは、崩壊監視の実務的な手本になっています。標準偏差が `1/√d`(d は次元数)に近い値で安定していれば健全、ゼロへ向かえば崩壊、という見方を提供しました。
- VICReg (Bardes et al., 2022): ImageNet linear probe で BYOL/SimSiam に匹敵する Top-1(約 73% 前後)を、対称な2ブランチ + 正則化のみで達成。variance 項を外すと即崩壊することをアブレーションで示し、分散保持が崩壊防止に直結することを定量化しました。covariance 項を外すと次元崩壊が進み精度が低下することも示されています。
- I-JEPA (Assran et al., 2023): ViT-Huge で ImageNet linear probe を競争力ある精度で達成しつつ、画素再構成系(MAE 等)より少ない事前学習計算で良い表現を得られると報告。特徴空間での予測 + EMA ターゲット + stop-gradient という JEPA の組み合わせが大規模でも崩壊せず機能することを示しました。

これらに共通する教訓は2つです。第一に、崩壊は損失だけ見ると「順調に下がっている」ように見えるため気づきにくく、出力の分散(標準偏差)や下流タスク精度で別途監視する必要があること。第二に、非対称性(stop-gradient / EMA / predictor)か分散正則化のいずれか、崩壊を封じる仕組みを必ず1つは入れないと、joint-embedding 系は自明解へ落ちる、ということです。

## メリット・トレードオフ・限界

メリット
- frozen / stop-gradient は実装が単純(`.detach()` や `requires_grad=False`)で、崩壊を確実に防げる。frozen はバックプロップしないので計算も軽く、ターゲットが定常で学習が安定する。
- EMA はターゲットも徐々に良くなる柔軟さがあり、VICReg は対称構造のまま崩壊を防げる。
- 幾何ワープによるアンカーは固定ターゲットと別経路の防御で、併用すると頑健性が増す。入力依存の構造を予測に強制できる。

トレードオフ・限界
- 完全固定のターゲットは「事前学習時点の特徴空間」に縛られ、タスクへ最適化されにくい。EMA は柔軟だが非定常で崩壊リスクと設定依存(τ)が増す。
- 崩壊は損失上は見えないため、分散や下流性能で監視しないと見逃す。部分崩壊(次元崩壊)はさらに発見しづらく、有効次元(特徴共分散の固有値スペクトル)を測る必要がある。
- VICReg は正則化項3つの重み調整が要る。SimSiam は predictor の設計や学習率に敏感。
- ワープアンカーは座標系整合(別項)を正しく実装してこそ効く。整合を誤ると逆効果になりうる。
- 研究上の未解決課題: 「なぜ非対称構造だけで崩壊しないのか」の完全な理論的説明はまだ発展途上で、predictor が固有空間を整える役割、バッチ正規化が暗黙に分散を確保する役割、stop-gradient が暗黙的に交互最適化を行う役割など複数の仮説が並立しています。

## 発展トピック・研究の最前線

- dimensional collapse の解析: 出力共分散行列の固有値が一部ゼロへ潰れる「次元崩壊」は、完全崩壊より検出が難しく、特徴の有効次元(effective rank)を測る研究が進んでいます(Jing et al., 2022)。固有値スペクトルの監視が、健全性チェックの新しい標準になりつつあります。
- 理論的理解: predictor とバッチ正規化が EMA の役割を代替しうる、stop-gradient が暗黙的に交互最適化(期待値最大化に似た反復)を行っている、といった理論研究が崩壊回避の本質に迫っています。
- 大規模・マルチモーダルへの展開: I-JEPA に続く V-JEPA(動画)など、JEPA 系は時空間・マルチモーダルへ拡張中で、崩壊防止の原理(stop-gradient + EMA ターゲット)はそのまま受け継がれています。動く自車のような幾何が絡む場面では、ワープアンカーのような問題固有の防御を組み合わせるのが最前線の工夫です。
- 監視の自動化: 学習中に標準偏差や有効次元を継続監視し、崩壊の兆候(分散の急減)を検知したら学習率や正則化重みを自動調整する、といった運用面の工夫も実用上重要なテーマです。
