# SOLOFusion (長期時間融合)

## ひとことで言うと

カメラだけで物体の3D位置、特に「どれくらい遠いか」という奥行きを当てるのは原理的に難しいのですが、複数の時刻のフレームをうまく組み合わせると精度が上がります。SOLOFusion は「短い時間の高解像度な情報」と「長い時間の低解像度な情報」を両方使うことで、カメラのみの BEV(真上視点)3D 物体検出を大きく改善した時間融合(temporal fusion)の研究です。論文タイトルの "Time Will Tell"(時間が教えてくれる)は、長い時間を見ることの効果を強調したものです。

## 直感的な理解

なぜ時間を使うと奥行きが分かるのか。1枚の写真は3次元の世界を2次元に潰したものなので、「小さく写った近くの物」と「大きく写った遠くの物」が同じ見た目になりえます。だからカメラ1枚から距離を当てるのは本質的に曖昧です。

ここで助けになるのが「カメラ自身が動いている」という事実です。車に積まれたカメラは時間とともに移動するので、少し時間が経つと同じ物体が違う角度・違う位置から写ります。人間が片目をつぶっても、頭を左右に少し動かせば遠近感がつかめるのと同じ原理です。これを運動視差(motion parallax)と呼びます。視点が動いた量が分かっていれば、2つの時刻の見え方の違いから、三角測量のように奥行きを逆算できます。これが時間融合の基本アイデアです。

問題は「どれくらい長く、どれくらい細かく過去を見るか」です。SOLOFusion 以前の研究は、たいてい直前の1〜2フレーム程度しか使っていませんでした。SOLOFusion の出発点となった観察が、この章で深掘りする2点です。短い履歴では視点の移動量が小さく、特にゆっくり動く物や止まっている物の奥行きが出にくい。だが直前の高解像度な情報は速い動きや細かい位置合わせに効く。両方欲しいが全部を高解像度で長く持つと計算が爆発する。このジレンマをどう解くか、が本研究の核心です。

## 基礎: 前提となる概念

理解の土台として、まずコストボリューム(cost volume)を説明します。これは時間融合で奥行きを当てる定番の道具です。ある画素について「奥行きが d だと仮定したら、別の時刻のフレームの対応位置の見た目とどれくらい一致するか(コスト=不一致度)」を、いろいろな奥行き候補 d(例えば 1m, 2m, …, 60m)について並べた3次元のデータです。一致度が高い(コストが低い)奥行きが正解に近い、という形で奥行きを推定します。ステレオカメラ(2眼)で奥行きを出すのと同じ考え方を、時間方向(動く1台のカメラの過去フレーム)に適用したものです。

次に基線長(baseline、ベースライン)です。三角測量では、2つの視点の間の距離が長いほど奥行きの推定が正確になります。ステレオカメラでは2眼の物理的な間隔が基線長ですが、時間融合では「ある時間に車がどれだけ移動したか」が基線長になります。短い時間しか見ないと基線長が短く、奥行きの分解能が落ちます。これが「長く見たい」動機です。

そしてエゴモーション(ego-motion、自車の動き)です。過去フレームの情報を現在のフレームに重ね合わせるには、その間に車がどれだけ動いたか(並進と回転)を知り、過去の BEV 座標を現在の座標系に変換する必要があります。この補正には自己位置推定(オドメトリや IMU)が使われます。エゴモーションが正確でないと、過去の情報が現在とズレた場所に重なり、融合がかえって精度を落とします。

最後に NDS と mAP という評価指標です。mAP(mean Average Precision)は検出の当たり具合を測る標準指標。NDS(nuScenes Detection Score)は nuScenes 独自の総合スコアで、mAP に加えて、位置・サイズ・向き・速度・属性の誤差をまとめて評価します。どちらも高いほど良い検出です。

## 仕組みを詳しく

SOLOFusion の核心は「短期高解像度 + 長期低解像度」のハイブリッド時間融合です。2つの経路に分けて説明します。

短期(short-term)経路は高解像度・少フレームです。直前の数フレームについて、高い解像度でコストボリュームを作ります。視点の移動が小さくても解像度が高いので細かい位置合わせができ、速い動きや近距離の物に効きます。ただし基線長が短いので、これだけでは遠くやゆっくりの物の奥行きが弱い。

長期(long-term)経路は低解像度・多フレームです。ずっと長い履歴(例えば16フレーム分)について、低い解像度の BEV 特徴を保持して融合します。低解像度なので長期間ぶんを持ってもメモリ・計算が現実的に収まります。長く見ることで基線長が大きくなり、止まっている・ゆっくりの物の奥行きが効くようになります。

効率の決定的な工夫が「再帰的(リカレント)な履歴の持ち越し」です。長期の履歴を毎フレーム「過去16枚ぶん全部もう一度計算する」のは無駄です。SOLOFusion は前フレームで作った BEV 特徴を、エゴモーションに合わせて座標変換して現在のフレームに持ち越し、新しいフレームの情報だけを足し込みます。RNN(再帰ニューラルネット)が状態を持ち越すのと同じイメージです。これにより、長い履歴を使っても1フレームあたりの追加計算をほぼ一定に抑えられます。仮に毎フレーム履歴を再計算すると計算量は履歴長に比例して増えますが、再帰方式なら履歴長によらずほぼ一定になります。

全体の流れを数値イメージで示します。

```
各時刻: カメラ画像 → 画像特徴
   │
   ├─ 短期経路: 直前 数フレーム・高解像度
   │   → コストボリューム → 細かい奥行き手がかり(速い物・近い物)
   │
   └─ 長期経路: 過去 約16フレーム・低解像度BEV
       → エゴモーション補正して再帰的に蓄積・融合
       → 大きな基線長による奥行き手がかり(遅い物・遠い物)
   │
   ▼ 合体したBEV特徴
3D検出ヘッド(中心位置・サイズ・向き・速度を回帰)
```

## 手法の系譜と主要論文

カメラのみ BEV 3D 検出の時間融合は、数年で急速に成熟しました。系譜を追います。

時間方向を最初に明確に取り込んだ初期手法が BEVDet4D(Huang & Huang, 2022, arXiv:2203.17054)です。単フレームの BEVDet に直前フレームを足し、特に速度推定を改善しました。ただし使うのは直前の1フレーム程度で、時間の活かし方は限定的でした。SOLOFusion は「もっと長く見ればまだ良くなる」ことを実証してこの路線を発展させた、と位置づけられます。

別アプローチが BEVFormer(Li et al., ECCV 2022, arXiv:2203.17270)で、時空間アテンションにより複数時刻を融合します。ただし扱う履歴は短めでした。SOLOFusion はコストボリューム + 長期再帰という幾何ベースの別アプローチで時間情報を取り込みます。

奥行きを別の方向から改善したのが BEVDepth(Li et al., AAAI 2023, arXiv:2206.10092)です。LiDAR を教師にして奥行き推定を明示的に監督し、BEV 検出を改善しました。これは SOLOFusion の「時間による奥行き改善」と相補的で、両者を組み合わせる研究もあります。

そして SOLOFusion(Park et al., ICLR 2023, arXiv:2210.02443)です。提案は「短期高解像度 + 長期低解像度のハイブリッド時間融合と、再帰的な長期履歴保持」。動機は、時間融合における奥行き精度が履歴の長さと解像度の両方に依存し、両者の得意が違うことでした。シンプルながら強力で、後続研究の標準ベースラインとして広く使われました。この後、StreamPETR や SparseBEV のように、長期時間情報を疎なクエリ表現で持つ流れが主流化していきます。

## 論文の実験結果(定量データ)

評価は nuScenes(屋外運転の大規模公開データ、約1000シーン、6カメラ + LiDAR)のカメラのみ3D検出で行われました。指標は前述の mAP と NDS です。

報告によると、SOLOFusion は当時の最先端を大きく更新し、NDS と mAP を当時のトップ水準に押し上げました。具体的には、テストセットの NDS でおよそ0.54前後という、当時のカメラのみ手法として最高クラスのスコアを達成したと報告されています(NDS は0〜1で、0.5を超えるとカメラのみとしては非常に強い水準)。単フレームや短期融合のベースラインと比べて、mAP・NDS ともに明確に上回りました。

アブレーションが本研究の主張の心臓部です。論文は履歴の長さを変える実験を行い、(1) 履歴を長くするほど検出精度が上がり、特に2フレームから16フレーム規模へ伸ばすと NDS が一貫して向上することを示しました。これが "Time Will Tell" の根拠です。(2) 短期(高解像度)経路を抜くと、速い物や近距離の精度が落ちました。(3) 長期(低解像度)経路を抜くと、ゆっくりの物や遠距離、特に静止物の奥行き精度が落ちました。両者の得意分野が違い、両方が必要だという主張を定量的に裏づけています。さらに (4) 再帰的持ち越しを使うことで、長期履歴を使っても1フレームあたりの計算が抑えられ、精度と効率を同時に達成できることが示されました。これらの結果は、その後の時間融合研究において「どれだけ長く見るか」を主要な設計軸として定着させました。

## メリット・トレードオフ・限界

メリットは、短期(高解像度)と長期(低解像度)の使い分けにより、近距離・速い物も遠距離・遅い物も奥行きを当てやすいこと、再帰的な履歴保持により長い時間を使っても1フレームあたりの計算がほぼ一定で効率的なこと、カメラのみで LiDAR に迫る奥行き精度を低コストで得られること、構造がシンプルで多くの後続研究のベースラインになったことです。

トレードオフと限界もあります。長期融合がエゴモーション補正に依存するため、自己位置推定の誤差が積もると精度が落ちます。動的物体は時間とともに動くので、静止を前提としたエゴモーション補正だけでは長期履歴の位置合わせにズレが生じえます(速い物は長期経路の恩恵を受けにくい)。履歴を持つぶん実装が複雑で、推論時にリカレント状態の管理が必要です。低解像度の長期 BEV は細部を犠牲にするので、小さい物体や混雑シーンでの分解能には限界があります。評価が nuScenes 中心であり、フレームレートやカメラ配置が異なる環境への移植は要検証です。

## 発展トピック・研究の最前線

研究の最前線は、SOLOFusion が確立した「長く見るほど良い」という知見を、より効率的・柔軟に実現する方向に進んでいます。第一に疎なクエリベースの時間融合で、StreamPETR や SparseBEV は、密な BEV 特徴を長期保持する代わりに、物体クエリに時間情報を伝播させることで、メモリを抑えつつ長期履歴の恩恵を得ます。第二に時間融合と奥行き監督の統合で、BEVDepth 系の明示的奥行き監督と SOLOFusion 系の時間三角測量を組み合わせ、両者の強みを足す研究があります。第三に動的物体への対応で、エゴモーション補正だけでは追えない動く物体の長期一貫性を、運動補償付きのワーピングや物体中心の時間モデリングで扱う試みが進んでいます。SOLOFusion は「時間は奥行きの強力な手がかりであり、長期と短期は補完的」という基本原理を確立した点で、カメラ BEV 検出の時間モデリングの基礎文献であり続けています。
