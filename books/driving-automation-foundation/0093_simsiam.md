# SimSiam

## ひとことで言うと
SimSiam(シムシャム、Simple Siamese)は、ラベルのない画像から良い特徴を学ぶ自己教師あり学習の手法であり、同時に「崩壊(全部同じ答えに潰れる失敗)を防ぐには、いったい何が最低限必要なのか」を引き算の実験で突き止めた研究です。結論は驚くほど簡素で、負例(比較対象)も EMA(ゆっくり更新する別のコピー)も大きなバッチも不要で、小さな予測器(predictor)と stop-gradient(勾配停止)さえあれば崩壊を防げる、というものでした。

## 直感的な理解
自己教師あり学習の中心アイデアは「同じ画像から少し違う2枚(ビュー)を作り、その特徴を互いに近づければ、画像の本質を捉えた表現が育つ」というものです。問題は、これだけだと「全部の画像に同じ定数を返す」という究極の手抜きで損失がゼロになってしまうこと(崩壊)です。

それまでの研究は、崩壊を防ぐためにさまざまな「お守り」を付けていました。対照学習(contrastive learning)は大量の負例(違う画像と引き離す相手)、BYOL は EMA ターゲット、SwAV はクラスタリングと均等割り当て、といった具合です。SimSiam の問いは「これらのお守りを1つずつ外したら、どこで崩壊するのか」でした。お守りを全部外していくと、最後に残った最小の組み合わせ(predictor + stop-gradient)で崩壊が止まりました。これは「崩壊回避の必要最小要件は何か」という基礎科学的な問いに、実験で答えを出したことを意味します。物理で言えば、複雑な装置から部品を1つずつ外して「どの部品が本当に効いているか」を特定する、対照実験(コントロール)の自己教師あり学習版です。

## 基礎: 前提となる概念
Siamese(シャム、双子)ネットワークとは、同じ重みを共有する2つの枝(branch)で2つの入力を処理する構造です。重みが共有されているので、本質的には1つのネットワークに2回入力を通すのと同じですが、2つの出力を比較する点が肝です。元々は署名照合や顔照合のように「2つが同じか」を測る用途で生まれた構造です。

データ拡張(data augmentation)とは、1枚の画像にランダムな変換(切り抜き crop、色変化 color jitter、左右反転 flip、グレースケール化、ぼかしなど)を施し、見た目は違うが中身は同じ複数の画像を作る操作です。SimSiam はこれで2つのビューを作ります。何を不変とみなすか(色を変えても同じ犬、切り抜いても同じ犬)を拡張の選び方で教えている、と捉えると本質が見えます。

predictor(予測器)とは、片方の枝の出力にだけ挟む小さな MLP(多層パーセプトロン、全結合層を重ねた小さなネットワーク)です。SimSiam では入力 2048 次元 → 中間 512 次元 → 出力 2048 次元のボトルネック型 MLP が使われます。中間を細くする(ボトルネック)ことが安定に寄与すると報告されています。

projection head(射影 MLP)とは、バックボーン(ResNet-50 など)の出力に重ねる別の MLP で、損失を計算する空間へ写す役割です。下流タスクには projection head を外したバックボーン特徴を使うのが通例です。

stop-gradient(勾配停止)については [Stop-gradient (勾配停止)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0658_stop-gradient) を参照。ターゲット側に勾配を流さず、固定された的とみなす操作です。

崩壊(collapse)は、入力に関係なく同じ定数を出す失敗。出力の標準偏差がゼロに張り付くのが目印です。

## 仕組みを詳しく
SimSiam の1ステップは次の通りです。

1. 1枚の画像 x にデータ拡張を2回かけ、ビュー x1, x2 を作る。
2. 重み共有のエンコーダ f(バックボーン ResNet-50 + projection head)に通し、特徴を得る。f(x1) = z1, f(x2) = z2。どちらも形状 `[N, 2048]`(N はバッチサイズ)。
3. 片方に predictor h を通す。p1 = h(z1)、形状 `[N, 2048]`。
4. p1 を z2 に近づけるよう学習する。ただし z2 には stop-gradient をかける(z2 = 固定された目標、勾配を流さない)。
5. 対称に p2 = h(z2) を sg(z1) に近づける。
6. 損失は両方向の平均:

```
loss = D(p1, sg(z2)) / 2  +  D(p2, sg(z1)) / 2
```

D は負のコサイン類似度: `D(p, z) = -(p/||p||) · (z/||z||)`。揃うほど -1 に近づき小さくなります。

SimSiam に「無い」ものを並べると設計の簡素さが際立ちます。
- 負例が無い(違う画像と遠ざける処理をしない)。対照学習で必須の大量の負例が不要。
- EMA で更新する別のターゲットネットワークが無い(両枝は同じ重みを共有)。BYOL との最大の差。
- 大きなバッチが必須でない(バッチ 256 でも安定。SimCLR が要する数千の大バッチは不要)。

それでも崩壊しません。論文の核心実験は、stop-gradient を外した瞬間に挙動が一変することです。stop-gradient ありなら出力の各次元の標準偏差は `1/sqrt(d)`(d は次元、L2 正規化された出力が単位球上に一様に散らばるときの理論値)付近で安定しますが、外すと標準偏差がゼロに急落し、損失は最小値 -1 へ張り付きます。これが完全崩壊の指紋です。つまり崩壊回避の主役は stop-gradient であり、predictor がそれを補助する、という分離が示されました。

なぜ効くのかについて、論文は「期待値最大化(EM, expectation-maximization)に似た交互最適化」という仮説を提示しました。損失を、各画像ごとの潜在的な「代表ベクトル」を導入した2変数の目的関数とみなすと、stop-gradient で片側を固定して他方を合わせる手続きが、EM の E ステップ(代表を固定して割り当てを更新)と M ステップ(割り当てを固定して代表を更新)の交代に対応する、という説明です。論文はこの仮説を補強するため、stop-gradient の代わりに「複数ビューの平均で代表を明示的に推定する」変種でも学習が進むことを示しました。後続の DirectPred(Tian et al., ICML 2021)はこの力学をより厳密に解析し、predictor が表現の共分散行列の固有空間に整列することで崩壊が避けられると示しました。

predictor を外すとどうなるか。論文のアブレーションでは、predictor を恒等写像(何もしない)にすると即座に崩壊します。predictor の出力に BatchNorm を入れた状態が安定で、predictor を凍結(ランダム初期値のまま学習しない)すると収束しません。つまり predictor は「学習される変換」であることが重要だと分かります。また predictor の中間次元を入力次元と同じ 2048 にするより、512 に絞ったボトルネック構造のほうが安定すると報告されています。

BatchNorm の役割。projection head の各層と predictor の隠れ層に BatchNorm を入れることが安定に効きますが、predictor の出力層に BatchNorm を入れると不安定になります。論文は「BatchNorm は最適化を助けるが、崩壊回避そのものを担うのは stop-gradient であり BatchNorm ではない」ことを、BatchNorm の有無を変えた追加実験で確認しています。これは「BYOL が崩壊しないのは BatchNorm のおかげ」という当時の対抗仮説への反証にもなりました。

## 手法の系譜と主要論文
- SimCLR(Ting Chen et al., ICML 2020, arXiv:2002.05709)。同一画像の2ビューを近づけ、バッチ内の他画像を負例として遠ざける対照学習。崩壊は大量の負例で防ぐが、数千の大バッチが必要。
- MoCo(Kaiming He et al., CVPR 2020, arXiv:1911.05722)。負例をキュー(memory queue)に貯め、キー側エンコーダを EMA 更新。大バッチ不要にした。
- BYOL(Jean-Bastien Grill et al., NeurIPS 2020, arXiv:2006.07733)。負例を捨て、EMA ターゲット + predictor + stop-gradient で崩壊回避。「負例なしでも学べる」を示した転機。
- SwAV(Mathilde Caron et al., NeurIPS 2020, arXiv:2006.09882)。オンラインクラスタリングと均等割り当て(Sinkhorn-Knopp)で崩壊を防ぐ別系統。multi-crop 拡張を導入。
- SimSiam(Xinlei Chen & Kaiming He, CVPR 2021, arXiv:2011.10566)。上記の崩壊回避の道具(負例・EMA・大バッチ)を1つずつ削り、predictor + stop-gradient さえ残れば崩壊しないと示した。最小要件の特定が新規性。
- DirectPred(Yuandong Tian, Xinlei Chen, Surya Ganguli, ICML 2021, arXiv:2102.06810)。SimSiam/BYOL の力学を理論解析した後続研究。predictor の固有空間整列を解明。

## 論文の実験結果(定量データ)
評価指標は ImageNet 線形評価(特徴を凍結し線形分類器1層だけ学習して top-1 正解率を測る。ImageNet は約128万枚・1000クラスなのでチャンスレベルは約 0.1%)です。

SimSiam は ResNet-50・100 エポックの事前学習で ImageNet 線形評価 top-1 約 67.7% を報告。同条件(100エポック相当)で SimCLR 約 66.5%、MoCo v2 約 67.4%、BYOL 約 66.5%、SwAV 約 66.5% であり、SimSiam ははるかに単純な構成でこれらと同等以上でした。より長い学習でも、SimSiam は 800 エポックで約 71.3% に達し、競合手法と肩を並べます。要点は「最も部品が少ないのに性能が落ちない」ことです。なお BYOL や SwAV は学習エポックをさらに伸ばすと 74〜75% 台に届くため、最高到達点では上回られますが、構成の単純さあたりの効率では SimSiam が際立ちます。

決定的なアブレーション。stop-gradient を外すと、出力標準偏差が即座にゼロへ落ち(崩壊)、線形評価精度はチャンスレベル(約 0.1%)に転落します。「67.7% vs 約 0%」という落差が、stop-gradient の有無が表現の有用性を二分することを示します。

バッチサイズ依存性。SimSiam はバッチ 128〜512 で安定し、SimCLR のように数千の大バッチを必要としません。論文はバッチ 64〜4096 まで振り、64〜2048 で精度がほぼ平坦(67% 前後)、極端な大バッチ 4096 でやや低下することを示しました。これは負例を使わないため、バッチ内に多数の比較対象を集める必要がないことの直接の帰結です。

予測器(predictor)のアブレーション。predictor を恒等写像にすると崩壊。predictor を固定(ランダム初期値で学習しない)すると収束しません。一方、predictor の学習率を一定に固定する(コサイン減衰を適用しない)ほうがむしろ安定して精度が上がる、という細かい知見も報告されています。

転移学習(別タスクへの応用)。SimSiam で学んだ特徴は、物体検出(VOC, COCO)やインスタンスセグメンテーションへ転移したとき、教師あり ImageNet 事前学習と同等かそれ以上の検出性能(平均適合率 AP の指標)を示し、自己教師あり特徴の実用性が確認されました。COCO の検出・セグメンテーションで教師あり事前学習を上回るケースも報告されています。

## メリット・トレードオフ・限界
メリット
- 構成が非常にシンプル(負例なし・EMA なし・大バッチ不要)。実装と再現が容易。
- 計算・メモリコストが軽い。1枚の画像につき2ビュー、1つのネットワークだけで済む。
- 崩壊回避の必要最小要件を明らかにし、後続の理論研究(DirectPred 等)と統一的理解の出発点になった。

トレードオフ・限界
- なぜ崩壊しないかの理論は当時仮説段階(EM 解釈)で、厳密化は後続研究を待った。
- predictor の構造・学習率・正規化にやや敏感で、設定を誤ると崩壊や不収束に陥る。
- 対照学習のような明示的な「異なるものを引き離す」信号がないため、データ拡張の設計品質に性能が依存しやすい。拡張が弱いと意味のある不変性を学べない。
- 次元崩壊(出力が低次元部分空間に潰れる)を完全には保証せず、分散・脱相関の正則化([VICReg](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0661_vicreg_note) / [Barlow Twins](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0618_barlow-twins))と相補的という見方がある。
- 最高到達点(十分長く学習したとき)では EMA や multi-crop を使う手法にやや劣る場合がある。単純さと最終性能のトレードオフ。

## 発展トピック・研究の最前線
SimSiam が示した「最小要件」は、その後の自己教師あり学習の統一理解を加速しました。対照(負例)・非対照(predictor + stop-gradient)・正則化(分散・共分散)という3系統が、いずれも「表現の分散を保ち次元間相関を減らす」という共通原理に還元できる、という議論です。また Transformer バックボーン(ViT)上での SimSiam 系の挙動、stop-gradient と BatchNorm/LayerNorm の相互作用、predictor を解析的に決める方法(DirectPred)、次元崩壊を共分散の固有値で測る研究(Jing et al., 2022)など、力学の理解が深まっています。実用面では、SimSiam の思想は画像だけでなく動画・点群・マルチモーダルへ拡張され、特徴空間で予測する世界モデル系の target encoder への stop-gradient 設計にも直系で受け継がれています。
