# Neural ODE / ODE-RNN

## ひとことで言うと
Neural ODE は、ニューラルネットワークの「層を積み重ねる」という離散的な発想をやめて、隠れ状態が「時間とともに連続的に変化していく」と考える手法です。状態の変化の速さをニューラルネットで表し、それを微分方程式の数値ソルバーで解いて答えを得ます。応用として ODE-RNN / Latent ODE があり、これらは「観測の時間間隔がバラバラな時系列(例: 患者の不定期な検査記録)」を、人工的な穴埋めなしに自然に扱えるのが大きな強みです。

## 直感的な理解
普通の RNN や多くの時系列モデルは「1 ステップ=固定の時間間隔」を暗黙に前提にします。ところが現実のデータは観測の間隔がバラバラなことが多いです。病院のカルテは検査が不定期(今日、3 日後、次は 2 週間後)、IoT センサーは時々通信が落ちて欠測する、金融取引は不規則な時刻に発生する、といった具合です。

固定間隔前提のモデルでこうした不等間隔(irregularly-sampled)データを扱うには、無理に時間軸を細かい等間隔に切って、観測のない時刻を補間や穴埋めで埋めるのが一般的でした。しかしこれには問題があります。穴埋めした値は本物ではないのにモデルは本物と区別できず、しかも時間軸を細かく切るほど大半が穴だらけの巨大行列になって非効率です。「2 週間空いた」という時間間隔そのものが持つ情報も失われがちです。

Neural ODE の発想は「状態は観測がない間も、物理現象のように連続的に動き続けている」と考えることです。患者の容態は検査していない間も変化し続けているはずで、その変化を連続時間の方程式でモデル化すれば、観測が来た瞬間だけ情報を取り込み、間は自然に時間発展させればよい、というわけです。これにより「2 週間空いた」も「3 時間空いた」も、積分する時間の長さの違いとして素直に扱えます。

## 基礎: 前提となる概念
まず普通のニューラルネット、特に ResNet(残差ネットワーク)の話から始めます。ResNet では各層が「前の層の出力に、ちょっとした変化分を足す」形をとります。

```
h_{t+1} = h_t + f(h_t, θ_t)
```

ここで h_t は t 層目の隠れ状態(中間表現のベクトル)、f はその層のニューラルネット、θ_t はその層のパラメータです。「前の値に変化分 f を足す」を何十層も繰り返します。

ここで微分方程式の基礎を噛み砕きます。常微分方程式(ODE, Ordinary Differential Equation)とは「ある量の変化の速さ(微分)が、その量自身と時刻でどう決まるか」を書いた式です。たとえば「貯金の増える速さ=金利×残高」のように、状態がどう動くかのルールだけを与え、初期値から出発して時間を進めれば軌跡が決まります。これを数値的に解く一番素朴な方法がオイラー法で、「今の変化の速さ × 小さな時間幅 Δt」を足し込む操作を繰り返します。

Chen らはここで本質的な気づきを得ました。ResNet の式は、まさにオイラー法の 1 ステップ(変化の速さ f に時間幅 1 をかけて足す)とそっくりなのです。ということは、層の数を無限に増やしてステップ幅を限りなく小さくしていくと、離散的な層の積み重ねは連続的な微分方程式に収束します。

```
dh(t)/dt = f(h(t), t, θ)        … 状態の「変化の速さ」をネットで表す
```

つまり「何層積むか」という離散ハイパーパラメータを決める代わりに、「状態の変化の速さ f」だけを 1 つのニューラルネットで定義し、あとは ODE ソルバー(常微分方程式を解く数値計算ライブラリ)に「t=0 から t=T まで積分して」と頼めば、出力 h(T) が得られる、という発想です。これが Neural ODE(Neural Ordinary Differential Equation、ニューラル常微分方程式)です。

## 仕組みを詳しく
3 つの段階に分けて説明します。

### 1. Neural ODE 本体(連続変換)
入力 h(0) を初期状態とし、変化率 f(h, t, θ) を定義する小さなニューラルネット(例: 数層の MLP)を用意します。出力は次の積分で得られます。

```
h(T) = h(0) + ∫_0^T f(h(t), t, θ) dt
```

これをソルバー(オイラー法、ルンゲ・クッタ法、適応刻みの Dopri5 など)で数値的に解きます。適応刻みソルバーは「変化が激しいところは細かく、緩いところは粗く」とステップ幅を自動調整し、必要な精度を保ちつつ無駄な計算を減らします。1 回の forward に必要なソルバーの評価回数を NFE(Number of Function Evaluations、関数 f を何回呼んだか)と呼び、これが Neural ODE の計算コストの実質的な指標になります。

学習(誤差逆伝播)には随伴感度法(adjoint sensitivity method)を使います。素朴にソルバーの全計算ステップを記録して逆伝播すると(これを通常のバックプロパゲーションでやる、と考えてください)、ステップ数に比例した膨大なメモリを食います。随伴法は、随伴状態 a(t) = ∂L/∂h(t)(損失 L が時刻 t の状態にどれだけ敏感かを表すベクトル)を導入し、これが da(t)/dt = −a(t)^T ∂f/∂h という別の ODE(随伴 ODE)に従うことを利用します。これを終端時刻 T から t=0 へ時間逆向きに解き、その途中でパラメータ θ に関する勾配を積分で蓄積します。こうすると中間ステップを保存せずに勾配を計算でき、メモリをほぼ一定(O(1)、積分区間の長さによらない)に抑えられます。これが「深さ(積分区間)を増やしてもメモリが増えない」という Neural ODE の実用上の決定的な利点です。代償として、逆向き積分は順方向の軌跡を再構成しながら解くため数値誤差が入りやすく、長期積分では誤差が蓄積します。これを嫌う場合は、順方向の状態を保存して厳密に逆伝播する方式(精度は高いがメモリを食う)とのトレードオフになります。

### 2. ODE-RNN(不等間隔時系列の処理)
時系列に使うときの基本形が ODE-RNN です。考え方はシンプルで、「観測と観測のあいだ」と「観測が来た瞬間」を別々に扱います。

```
h = 初期状態
for 各観測 (t_i, x_i) を時刻順に:
    # 前の観測時刻 t_{i-1} から今の t_i まで、ODE で状態を連続的に進める
    h = ODESolve(f, h, t_{i-1}, t_i)
    # 今の観測 x_i を RNN(GRU など)で取り込み、状態を書き換える
    h = GRUCell(h, x_i)
```

観測のない間は ODE が状態を物理的に時間発展させ、観測が来た瞬間だけ RNN 更新で情報を取り込みます。検査が 2 週間空いても、その間の容態変化を ODE が連続的にモデル化します。間隔の長短は ODESolve に渡す積分区間の長さとして自然に反映されます。

### 3. Latent ODE(生成モデル版)
Rubanova らは、潜在変数(latent、観測の裏にある本質的な状態)が 1 本の ODE に従うとする Latent ODE を提案しました。手順は、観測列を ODE-RNN でエンコードして初期潜在状態 z(0) の分布を推定し、そこから 1 本の ODE で潜在の軌跡 z(t) を生成、各観測時刻で z(t) をデコードして観測を再構成・予測します。これは VAE(変分オートエンコーダ)の枠組みに連続時間の潜在ダイナミクスを組み込んだもので、任意の時刻での予測や欠測の自然な補間、過去・未来両方向への外挿ができます。

数値例のイメージ: 患者 10 人の検査が各人バラバラな時刻に 5〜20 回ある不等間隔データ。従来は時間軸を 1 時間刻みなどに切って穴埋めし、巨大で穴だらけの行列にしていました。Latent ODE は実際の観測時刻(t=0.0, 3.7, 17.2, …)をそのまま積分区間に使うので、人工的な穴埋めが不要です。

## 手法の系譜と主要論文
- Chen, Rubanova, Bettencourt, Duvenaud, "Neural Ordinary Differential Equations"(NeurIPS 2018 Best Paper, arXiv:1806.07366)。提案: ResNet の連続極限として ODE を捉え、随伴法でメモリ一定の学習を実現。動機: 層数という離散ハイパーパラメータを連続化し、メモリ効率と連続時間モデリングを得るため。新規性: 連続深さの定式化と随伴法による O(1) メモリ学習。これが連続正規化フロー(密度推定)や不等間隔時系列への扉を開きました。
- Rubanova, Chen, Duvenaud, "Latent ODEs for Irregularly-Sampled Time Series"(NeurIPS 2019, arXiv:1907.03907)。提案: ODE-RNN と Latent ODE。動機: 医療など不等間隔・欠測の多い時系列を補間に頼らず自然に扱うため。新規性: エンコーダ(ODE-RNN)+ 連続潜在ダイナミクスという生成モデルの構成。
- Kidger, Morrill, Foster, Lyons, "Neural Controlled Differential Equations for Irregular Time Series"(NeurIPS 2020, arXiv:2005.08926)。提案: 観測を制御信号として連続経路に変換し、その経路で駆動される CDE(制御微分方程式)を解く。動機: ODE-RNN の「観測時に状態が離散ジャンプする」不自然さを、滑らかで理論的にきれいな枠組みに置き換えるため。新規性: 経路を連続関数として扱い、観測全体が連続的に状態を駆動する定式化(「連続時間版の RNN」として理論的に整理)。
- 周辺の発展として、ODE のべき乗的に増える NFE を抑える正則化(STEER, RNODE)、安定性を保証する設計、確率微分方程式版(Neural SDE)などが続きました。

## 論文の実験結果(定量データ)
評価には合成・実データの両方が使われます。代表的なベンチマークは PhysioNet 2012(ICU 患者の不等間隔な生理計測から院内死亡を予測)と MuJoCo(物理シミュレータから生成した不等間隔軌跡の補間・外挿)です。

まず元の Neural ODE 論文では、画像分類(MNIST)で、ODE ネットが同等精度の ResNet と肩を並べつつ、パラメータ数を約 1/3 に、メモリを随伴法によりほぼ一定(深さに依存しない O(1))に抑えられることが示されました。深さに相当する積分の細かさはソルバーが誤差許容に応じて自動調整するため、「何層積むか」を手で決める必要がなくなります。

不等間隔モデリングの効果は次のように示されました。Latent ODE / ODE-RNN は、観測時刻のあいだも状態を連続発展させるため、観測を等間隔に丸めて RNN に入れる素朴な手法より、補間(observed points を間引いて残りを当てる)や外挿(先の時刻を予測する)の誤差が小さいと報告されています。指標は再構成・予測の平均二乗誤差(MSE、予測値と実測値の差の二乗平均で、小さいほど良い)や、分類タスクでの AUC(ROC 曲線下面積、識別性能を 0〜1 で表し 1.0 が完璧で 0.5 が偶然)などです。MuJoCo の物理軌跡では、観測を間引くほど(=より不等間隔・疎になるほど)Latent ODE の優位が広がり、標準 RNN との MSE 差が拡大することが報告されました。PhysioNet 2012 の院内死亡予測(8000 人規模の ICU 患者、最初の 48 時間の不定期な計測から予測)では、時刻情報を正しく使うモデルが、時刻を無視して観測を等間隔列とみなすモデルより AUC で上回ることが示されました。

重要なアブレーションは「観測の間隔(時刻情報)を使うか否か」です。標準 RNN に時間間隔を陽に与えない場合、不等間隔データで性能が落ちますが、ODE で間を埋めるか、少なくとも時間差を入力に加えると改善します。これは「連続時間の扱いそのものが効いている」ことの切り分けです。一方で、データが実は等間隔に近い場合は、ODE の連続性の恩恵は小さく、計算コストだけが増えるという報告もあり、Neural ODE の利点は「真に不等間隔・連続力学が支配的なデータ」で顕著になることが分かっています。

計算コストの定量も示されています。随伴法はメモリをほぼ一定に保てる一方、学習が進むにつれてダイナミクス f が複雑化し、適応刻みソルバーが要求する NFE が増えて推論・学習が遅くなる現象が報告されました。STEER や RNODE などの正則化は、この NFE 増加を抑えて学習を数割高速化できると報告しています。Neural CDE は PhysioNet/Speech Commands の不等間隔設定で ODE-RNN 系を上回る精度を示し、理論的なきれいさと実性能を両立しました。

## メリット・トレードオフ・限界
メリット:
- 不等間隔・欠測の時系列を、人工的な穴埋めなしに自然に扱えます。時間間隔そのものを情報として活かせます。
- 任意の実数時刻で状態を評価でき、連続的な予測・補間・外挿ができます。
- 随伴法により、積分区間(実質的な深さ)を増やしてもメモリがほぼ一定です。

トレードオフ・限界:
- ODE ソルバーの呼び出し(NFE)が重く、特に適応刻みで高精度を要求すると学習・推論が遅くなりやすいです。NFE は学習が進むほど増えがちです。
- 力学が複雑だと数値が不安定になり、ソルバーが極端に細かいステップを刻んで失速・発散することがあります。安定化のための正則化や設計上の工夫が必要です。
- 随伴法の逆向き積分は数値誤差を持ち込み、長期積分では誤差が蓄積しえます。厳密さが要る場面では中間状態を保存する方式とのトレードオフになります。
- データが実質等間隔なら、複雑さに見合う利得が出ないことがあります。安全クリティカルな車載リアルタイム推論にそのまま使うには、速度・安定性の十分な検証が前提になります。

## 発展トピック・研究の最前線
Neural ODE は「連続深さ」「連続時間ダイナミクスの学習」という発想を機械学習に持ち込み、多くの派生を生みました。Neural CDE は不等間隔時系列の理論的に最もきれいな定式化として広く参照されます。確率を扱う方向では Neural SDE(確率微分方程式)が、ノイズや不確実性のある力学・生成モデルに連続時間の枠組みを与えました。密度推定では連続正規化フロー(FFJORD)が ODE を使って柔軟な確率分布を学習します。物理との接続も活発で、エネルギー保存などの構造を組み込む Hamiltonian / Lagrangian Neural Networks、既知の物理方程式と学習を融合する物理インフォームド手法へとつながっています。さらに、拡散モデルの連続時間定式化(確率フロー ODE)や、状態空間モデル([S4 (Structured State Space)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0501_state-space-model-s4) や [Mamba (Selective SSM)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0491_mamba-selective-ssm))との関係(どちらも連続時間の線形力学を出発点に持つ)も、連続時間系列モデリングという大きな潮流の中で活発に研究されています。
