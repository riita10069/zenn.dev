# DINO (自己蒸留)

## ひとことで言うと
DINO は、正解ラベルを一切使わず、画像だけから「物の形や境界をよく捉えた強い特徴」を学習する自己教師あり学習(self-supervised learning)の手法です。鍵は「生徒ネット」と「教師ネット」という2つのほぼ同じネットワークを用意し、生徒が教師の出力を真似るよう学習する自己蒸留(self-distillation)という枠組みです。教師は生徒の重みをゆっくり平均したものなので、自分で自分を教える形になります。学習後の Vision Transformer は、教えていないのに物体の輪郭を捉える注意(attention)を自然に獲得します。DINO は DIstillation with NO labels の略です。

## 直感的な理解
子どもが言葉を覚える前に、見た物の「同じもの・違うもの」を自然に区別できるようになるのを思い浮かべてください。「これは犬」というラベルを誰かに教わらなくても、形・質感・動きの共通性から「同じ種類のもの」をまとめる感覚が育ちます。DINO がやりたいのもこれです。人間がラベルを付けた巨大データに頼らず、画像そのものの構造から「良い特徴」を学びたいのです。

なぜラベルなしにこだわるのか。ラベル付けは高コストで、しかもラベルの種類(犬・猫・車…)に表現が縛られます。「車」とだけ教わったモデルは、車の細かな質感や周囲との関係といった、ラベルに現れない情報を捨ててしまいがちです。ラベルなしで学べば、データの量を桁違いに増やせ、特定タスクに縛られない汎用的な特徴が得られます。自動運転のように夜・雨・逆光・見たことのない物体に出会う領域では、この汎用性が効きます。

## 基礎: 前提となる概念

自己教師あり学習(self-supervised learning, SSL)とは、正解ラベルを人間が付けるのではなく、データそのものから自動で「問題と答え」を作って学習する方式です。たとえば「画像の一部を隠して当てさせる」「同じ画像の2つの加工版を近づける」といった問題を自前で作ります。

対比学習(contrastive learning)は SSL の代表的な一派で、「同じ画像の別バージョン(正例ペア)同士は近く、違う画像(負例)は遠くに」と特徴空間で押し分ける方式です。MoCo(He et al., 2020)や SimCLR(Chen et al., 2020)が有名です。しかし「違う画像(負例)」を大量に集める必要があり、バッチサイズを巨大にしたりメモリバンク(過去の特徴を貯めておく仕組み)を使ったりと扱いが面倒でした。SimCLR は性能を出すのに 4096 という巨大バッチを要した、というのがこの面倒さの典型例です。

崩壊(collapse)とは、ネットワークが「どんな入力に対しても同じ出力を返す」というズルに陥る現象です。同じ画像の2バージョンを近づけるだけなら、全部を同じ定数ベクトルにすれば損失はゼロになりますが、特徴は何の情報も持たなくなります。対比学習は負例でこれを防ぎますが、負例なしで崩壊を防ぐのが SSL の難所でした。

BYOL(Grill et al., 2020)は「負例なしでも、教師-生徒の自己蒸留だけで崩壊せずに学べる」ことを示しました。DINO はこの流れを Vision Transformer(ViT、画像を小さなパッチに分けて Transformer で処理するモデル。Dosovitskiy et al., 2021)に適用し、崩壊を防ぐ仕掛けを整理した手法です。

蒸留(distillation)はもともと「大きな教師モデルの知識を小さな生徒モデルに移す」技術(Hinton et al., 2015)ですが、DINO では教師と生徒が同じ大きさで、教師が生徒の過去の平均というのが特徴です。だから「自己蒸留」と呼びます。

## 仕組みを詳しく

### 教師-生徒の自己蒸留

2つの同じ構造のネットを用意します。

- 生徒ネット `s`: 普通に勾配で学習する。
- 教師ネット `t`: 勾配では学習しない。代わりに生徒の重みの EMA(指数移動平均、Exponential Moving Average=過去の重みをゆっくり平均した値)で更新する。更新式は `θ_t ← m·θ_t + (1-m)·θ_s`(m は 0.996〜1.0 に近い値で、学習が進むにつれ 1 に近づけるスケジュール=cosine schedule を使う)。教師は生徒の「ゆっくり版・安定版」であり、毎ステップ激しく動く生徒より安定したターゲットを提供します。興味深いことに、この EMA 教師は生徒よりわずかに精度が高くなる傾向があり、一種のモデルアンサンブル(複数時点の重みの平均)として働いていると解釈されます。

学習手順:

1. 1枚の画像から複数のビュー(view、切り抜き・色変換・ぼかしなどで作った別バージョン)を作る。大きく切り抜いた global view 2枚(画像の 50% 以上をカバー)と、小さく切り抜いた local view 数枚(計 8 枚程度、画像の 50% 未満)。これをマルチクロップ(multi-crop)と呼ぶ。
2. 生徒には全ビュー、教師には global view だけを入力する。
3. それぞれの出力を確率分布(softmax で K 次元のベクトル、K は例えば 65536)に変換する。
4. 生徒の各ビューの出力分布を、教師の global view の出力分布に近づける(クロスエントロピーで一致させる)。小さく切り抜いた local view からでも「画像全体を見た教師」と同じ答えを出させることで、「部分から全体を推測する」能力が育つ。

```
画像 ──┬─ global view ─→ 教師(EMA) ─→ 分布 P_t  (目標, stop-grad)
        ├─ global view ─→ 生徒 ──────→ 分布 P_s ─┐ クロスエントロピーで
        └─ local view  ─→ 生徒 ──────→ 分布 P_s ─┘ 近づける
教師の重み ← 生徒の重みの EMA(勾配は流さない)
```

### 崩壊を防ぐ2つの仕掛け: センタリングとシャープニング

自己蒸留には「全画像で同じ出力を返せば損失ゼロ」という崩壊の罠があります。DINO はこれを教師側の2操作で防ぎます。

- センタリング(centering): 教師の出力(softmax 前のロジット)からバッチ全体の平均(center `c`)を引く。`c` も EMA で更新する。これは特定の1次元に全出力が偏る崩壊(=その次元だけ常に高い)を防ぎ、出力を「均一方向(一様分布)」へ押す力になる。
- シャープニング(sharpening): 教師側の softmax の温度 `τ_t`(temperature、分布の鋭さを決める割り算の係数)を低く(例 0.04〜0.07)して分布を尖らせる。温度が低いほど最大値が強調され分布が尖る。これは出力が「全次元ぼんやり一様」になる崩壊を防ぎ、出力を「尖らせる方向」へ押す力になる。

センタリング(均一化)とシャープニング(尖鋭化)が逆向きの力で釣り合うことで、崩壊せずに意味のある表現が育ちます。どちらか一方だけだと、片方の崩壊モード(定数化、または一様化)に落ちます。負例を一切使わないのがポイントで、これにより対比学習特有の大バッチ・メモリバンクが不要になります。

### 数値例・テンソル形状

- 入力 global view: `[3, 224, 224]`、local view: `[3, 96, 96]`(チャンネル3, 縦横)
- ViT-S/16 なら 224×224 を 16×16 パッチに分割 → 14×14=196 パッチ + CLS トークン(画像全体を要約する特別なトークン)= 197 トークン
- 各トークンは 384 次元(ViT-S の場合)。最終的に CLS トークンの特徴を取り出す。
- 射影ヘッド(projection head): 特徴(384次元)→ 3 層 MLP → L2 正規化 → `K=65536` 次元のロジット → softmax で分布
- 教師分布 `P_t` と生徒分布 `P_s` の間でクロスエントロピー `-Σ P_t log P_s`

### 創発した性質

DINO で学習した ViT は、CLS トークンの自己注意(self-attention)マップを可視化すると、ラベルなしで学んだのに前景の物体だけを綺麗にハイライトします。これは「物体のセグメンテーション(画素ごとにどの物体かを塗り分ける)に近い情報が、教師なしで自然に獲得された」ことを意味します。CNN(畳み込みニューラルネット)ベースの SSL では現れにくく、ViT + 自己蒸留の組み合わせで顕著に出る点が論文の大きな発見でした。論文では、この注意マップを使った教師なしオブジェクトセグメンテーションが、専用に学習した手法に匹敵する性能を出すことも示されました。

## 手法の系譜と主要論文

DINO は「負例を使う対比学習」から「負例を使わない自己蒸留」への流れの到達点の一つです。

対比学習の確立(2020): He らの MoCo、Chen らの SimCLR が、正例を近づけ負例を遠ざける枠組みで教師あり事前学習に迫る精度を達成。ただし大量の負例(巨大バッチやメモリバンク)が必要。

負例なしへの転換(2020-2021): Grill らの BYOL が EMA 教師 + predictor で負例なし学習を実現。Chen & He の SimSiam が「stop-gradient さえあれば EMA すら不要」と簡素化。崩壊回避の本質が「非対称性」にあることが明らかになった([Predictor (予測ネットワーク)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0084_predictor-network) 参照)。

ViT + 自己蒸留(2021): Caron らの DINO が ViT に自己蒸留を適用し、センタリング/シャープニングで崩壊を防ぎ、注意が物体を捉える創発性質を発見。

マスク画像モデリングとの融合(2022): Zhou らの iBOT が、DINO の自己蒸留に「マスクしたパッチの特徴を当てる」課題(BERT のような穴埋めの画像版)を組み合わせ、パッチレベル(画素近傍)の表現を強化。

大規模化(2023): Oquab らの DINOv2 が、データ厳選パイプライン・モデル蒸留・iBOT 統合により DINO を大規模化し、微調整なしで多タスクに効く汎用視覚バックボーン(backbone、特徴抽出の土台ネット)を実現。

特徴予測への展開(2023-): Assran らの I-JEPA が「EMA 教師がターゲット特徴を生成し、生徒(+predictor)がそれを予測する」という DINO/BYOL の発想を、画像のマスク領域の特徴予測へ進化させ、動画世界モデル(V-JEPA)の基盤になった。

## 論文の実験結果(定量データ)

### Caron et al., DINO (ICCV 2021)

- 線形評価(linear probing、凍結した特徴の上に線形分類器だけ学習)で、ImageNet 分類精度 ViT-B/8 でおよそ 80.1% を達成。同時期の自己教師あり手法を上回り、教師ありに迫った。線形評価は「特徴がどれだけ素直に分類に使えるか」を測る標準的な物差し。
- k-NN 分類(特徴の最近傍だけで分類、学習パラメータゼロ)で約 78.3% を報告。k-NN でこの精度が出るのは「特徴空間がクラスごとに綺麗にまとまっている」証拠で、特徴の質の高さを示す。線形評価と k-NN の差が小さいことも、特徴がよく整理されている証拠とされた。
- 教師なしのセグメンテーション(注意マップを物体マスクとして評価)でも、注意が前景を捉えることを定量・定性の両面で示した。DAVIS-2017 の動画オブジェクト追跡でも、特徴の最近傍マッチングだけで競争力ある結果を出した。
- アブレーション: センタリングとシャープニングのどちらか一方を外すと崩壊する(出力が定数化、または一様化)ことを示し、両者の釣り合いが必須であることを確認。マルチクロップ(local view)を外すと精度が数ポイント低下し、「部分から全体を推測する」課題が表現の質に効くことを示した。EMA の momentum を 0(=教師が即座に生徒に同期)にすると学習が崩壊することも示し、教師の安定性が本質的と確認した。

### Oquab et al., DINOv2 (2023)

- 提案: 約 1.42 億枚の厳選画像(LVD-142M、重複除去とクラスタリングで質を担保)で学習し、ViT-g(約 11 億パラメータ)まで規模を拡大。データ厳選・iBOT 課題・大規模教師からの蒸留を統合。
- 定量結果: 凍結特徴のまま(微調整なし)で、ImageNet 線形評価で 86% 超、深度推定(NYUv2 など)・セマンティックセグメンテーション(ADE20K)・画像検索で、当時の弱教師あり(CLIP=画像とテキストの対で学ぶ手法 など)を上回るか同等を報告。微調整なしでこの汎用性は事実上の標準バックボーン化を促した。小型モデル(ViT-S/B)は ViT-g からの蒸留で作られ、小さくても高精度を保った。
- トレードオフ: 学習に膨大な計算と厳選データのパイプラインが必要。

### BYOL (Grill et al., NeurIPS 2020) — 前身

- 負例なしのオンライン/ターゲット2ネット + EMA で、ImageNet 線形評価 74.3%(ResNet-50)を達成し、当時の対比学習(SimCLR 約 69%)を上回った。負例不要でも崩壊しないことを示し、DINO の土台となった。

### I-JEPA (Assran et al., CVPR 2023) — 発想の継承先

- DINO/BYOL の「EMA 教師がターゲットを生成し、生徒が予測する」発想を、画像のマスク部分の特徴を予測する方向へ進めた。ピクセルでなく特徴空間で予測する点が新しく、ピクセル再構成型の MAE(Masked Autoencoder)より少ない計算・少ないエポックで同等以上の線形評価精度を達成。動画世界モデル(V-JEPA など)の基盤になった。one_liner にある「JEPA ターゲット生成の発想に直結する前提技術」とはこのつながりを指す。

## メリット・トレードオフ・限界

メリット
- ラベル不要で、物体境界を捉える汎用的で転移性の高い特徴が得られる。
- 負例(違う画像)を集める必要がなく、対比学習より扱いやすい(大バッチ・メモリバンク不要)。
- 得られた特徴は微調整なしでも多くのタスクに効く(DINOv2 の k-NN・線形評価・密予測=画素ごとの予測)。
- EMA 教師が安定したターゲットを与え、生徒の急変に引きずられにくい。

トレードオフと限界
- 崩壊防止(センタリング・温度・EMA 係数・マルチクロップ)のチューニングが繊細で、設定を誤ると即崩壊する。
- マルチクロップ・大規模データで学習計算が重い。
- 自己教師ありゆえ、十分な教師ありデータがある狭いタスクでは最終精度が教師ありに劣ることもある。
- 何を学んでいるかの解釈は注意マップ頼みで、保証された意味づけではない。注意マップには高ノルムのアーティファクト(背景に現れる異常に強いトークン)が出ることが後に判明した。
- 未解決課題: 崩壊が防げる厳密な理論はいまも完全ではなく、非対称性・stop-gradient・EMA の役割の数理的理解は研究途上。

## 発展トピック・研究の最前線
- DINOv2 レジスタトークン(Darcet et al., ICLR 2024): 注意マップに現れる高ノルムのアーティファクトを、追加の「レジスタ」トークン(計算用の捨てトークン)で吸収し、密予測の品質と注意マップの解釈性をさらに上げる改良。
- 動画・3D への拡張: V-JEPA など時間方向への展開や、自己蒸留特徴を 3D 再構成・占有予測(occupancy prediction=空間が物体で埋まっているかを予測)の事前学習に使う流れ。
- ロボティクス・自動運転での事前学習: ラベルに縛られない汎用視覚特徴は、多様な走行環境への汎化に有利とされ、知覚バックボーンの事前学習に応用が広がっている。
- 自己蒸留と生成の橋渡し: I-JEPA → V-JEPA の系譜で、自己蒸留の「EMA 教師でターゲット特徴を作る」発想が世界モデルの予測目標生成([Predictor (予測ネットワーク)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0084_predictor-network))に直結している。これにより、DINO は単なる画像分類用 SSL を超えて、未来予測・世界モデルの基盤技術として位置づけが広がった。
