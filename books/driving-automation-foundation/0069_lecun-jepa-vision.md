# LeCunの自律機械知能構想

## ひとことで言うと
これは、深層学習の第一人者ヤン・ルカン(Yann LeCun)が2022年に発表した「人間や動物のように常識を持って自律的に行動する AI をどう作るか」という長期的な設計図です。主張の核心は、「未来のピクセルを描き出す生成型ではなく、抽象的な特徴を予測する予測型のワールドモデルを、複数の時間スケールにわたって階層的に積み上げるべきだ」というものです。JEPA という手法群はこの構想を具体化したものであり、I-JEPA・V-JEPA・V-JEPA 2 はその部分的な実証にあたります。

## 直感的な理解
赤ちゃんは生後数ヶ月で、物を放すと落ちる、見えなくなった物は消えていない(物の永続性)、押すと動く、といった物理の常識を、誰にも教わらず観察だけで身につけます。一方、現在の大規模言語モデルは何兆語ものテキストを読んでいても、コップを倒したら水がこぼれるという当たり前の物理を体得していません。テキストには書かれていないからです。

ルカンの問いはシンプルです。「動物や人間は、なぜあれほど少ない経験で、新しい状況にうまく対応できるのか」。彼の答えは「彼らは観察を通じて世界モデル(頭の中で未来をシミュレーションする仕組み)を作り、それを使って行動の結果を、実際に試す前に予測しているからだ」というものです。だから危険な行動を実際にやってみる前に「これはまずい」と分かる。AI にも同じ仕組みを持たせよう、というのがこの構想の出発点です。そしてその世界モデルは、ピクセルを描く生成型ではなく、意味を予測する予測型であるべきだ、というのが彼の強い主張です。

## 基礎: 前提となる概念
この構想を理解するには、現在の主流 AI の弱点を押さえる必要があります。

大規模言語モデル(LLM、ChatGPT のような文章生成 AI)は大量のテキストを学びますが、物理世界の常識を体得していません。テキストは世界の影にすぎず、影だけからは立体は復元できない、という比喩でルカンはこれを説明します。

強化学習(reinforcement learning、試行錯誤と報酬で学ぶ方式)は、現実で何千回・何万回も失敗しないと学べません(サンプル効率が悪い、と表現される)。自動運転のように失敗が許されない領域では、この「現実での大量失敗」が致命的に非現実的です。世界モデルがあれば、失敗を頭の中で予測して避けられます。

生成型(generative)と予測型(JEPA)の違いも前提です。生成型は未来のピクセルを全部描き出します。予測型は未来の「特徴(意味を凝縮した数値ベクトル)」だけを予測します。ルカンは、予測不能な細部(葉の揺れ、人の細かい動き)を全部描こうとする生成型は無駄が多くぼやける、と批判し、予測型を推します。

エネルギーベースモデル(energy-based model、EBM)も重要な前提です。これは「ある (入力 x, 出力 y) の組み合わせがどれくらいありえないか」を、エネルギーという1つのスカラー値で表す枠組みです(LeCun らが2006年のチュートリアルで体系化)。低エネルギー = もっともらしい、高エネルギー = ありえない、と読みます。確率分布として正規化する必要がないため、複数の正解がある(多峰的な)状況を扱いやすいのが利点です。ルカンは構想全体をこの EBM の言葉で統一的に定式化しています。

## 仕組みを詳しく
論文「A Path Towards Autonomous Machine Intelligence」では、自律機械知能を構成する複数のモジュールが提案されています。主要なものを順に説明します。

ワールドモデル(world model)は中心部品で、「今の状態 + 自分が取る行動」から「次の状態」を予測します。人間で言えば「この角を曲がったら何が見えるか」を頭の中で想像する力です。

アクター(actor)は、ワールドモデルを使ってゴールに近づく行動列を探して選ぶ部品です。「もしこう動いたら未来はこうなる」を何通りも頭の中で試し、一番良いものを選びます(これは MPC=モデル予測制御の発想と同じです)。

コスト/批評家(cost / critic)は、その状態が「どれくらい望ましくないか」を測る部品です。これが小さくなる方向にアクターが行動を選びます。コストには、設計者が固定で決める部分(衝突は悪い、など。intrinsic cost)と、経験から学習する部分(trainable critic)の両方があります。

知覚(perception)モジュールは、生のセンサー入力を世界モデルが扱える状態表現に変換します。そして全体を統括する設定器(configurator)が、今どのタスクをやっているかに応じて各モジュールを調整します(同じ世界モデルを別のゴールに使い回すための切り替え役)。

そして思想の中心が JEPA(Joint Embedding Predictive Architecture)です。未来を「ピクセル」ではなく「埋め込み(embedding)」の空間で予測します。

```
生成型: 入力 → 次フレームの全ピクセル(例 224*224*3 = 150,528 個)を予測
JEPA : 入力 → 次の状態の特徴ベクトル(例 1024 個)を予測
```

なぜ特徴空間なのか。未来は本質的に予測しきれない部分が多く、それを全部当てようとすると無駄が多く、結果がぼやけます(二乗誤差を最小にする予測は全候補の平均=ぼやけた絵になるため)。特徴空間なら「予測できない細部は捨て、意味のある情報だけ残す」ことができます。これが構想全体を貫く設計原理です。

H-JEPA(階層型 JEPA)は、H が Hierarchical(階層的)の略です。世界の予測には複数の時間スケールがあります。「次の0.1秒で歩行者がどう動くか」という細かい予測と、「この交差点を抜けて目的地に着く」という大まかな予測です。

```
高階層 JEPA  : 長時間・抽象的(目的地まで行く計画)
   ↑
低階層 JEPA  : 短時間・具体的(次の0.1秒の動き)
```

低い階層が短時間・細かい予測を、高い階層が長時間・抽象的な予測を担当し、階層を積み重ねます。これにより、長期計画を「抽象レベルでざっくり立て、必要な部分だけ細かく詰める」ことができます。人間が「まず駅に向かう」という抽象計画を立て、歩きながら細かい足の運びを決めるのと同じ構造です。重要なのは、上位の階層ほど時間方向に「粗く」(数秒〜数分を1ステップに)抽象化される点で、これにより長期計画の探索空間が小さく保たれます。

不確実性の扱いも重要です。未来が複数あり得る場合、predictor に潜在変数(latent variable、いわば「どの未来か」を選ぶ隠れたスイッチ)を入れて、複数の可能性を表現します。さらにルカンは、この潜在変数の情報量を制限する正則化(regularization)を強調します。潜在変数に自由を与えすぎると、モデルが潜在変数だけで答えをカンニング(予測対象の情報を潜在変数経由で丸写し)して崩壊するからです。エネルギーベースモデルの言葉で言えば、潜在変数の情報量制限は「低エネルギー領域が広がりすぎる(全部もっともらしいと言ってしまう)退化」を防ぐための正則化に対応します。

## 手法の系譜と主要論文
中心文献は Yann LeCun, "A Path Towards Autonomous Machine Intelligence"(2022, OpenReview に公開のポジションペーパー)です。これは実験論文ではなく、今後の AI の設計方針を述べた構想論文(ビジョンペーパー)です。提案は上記モジュール群と H-JEPA、エネルギーベースの定式化、潜在変数の正則化。動機は「生成型・強化学習中心の現状では常識を効率的に学べない」という問題意識でした。構想段階のため定量評価はなく、後続研究が部分的に実証していくことを前提にしています。理論的な背景には、LeCun らの長年のエネルギーベース学習の研究(2006年のチュートリアル)があります。

その第一歩が I-JEPA(Assran ら, Meta AI, CVPR 2023, [I-JEPA (Image JEPA)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0059_i-jepa_note))です。1枚の画像の一部を隠し、見える部分から隠れた部分の特徴を予測。生成型(MAE 等)より少ない計算で良い表現が学べることを示し、構想の核(特徴空間予測)を画像で実証しました。

続いて V-JEPA / V-JEPA 2(Meta AI, 2024-2025, [V-JEPA 2](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0101_v-jepa-2))が動画版を確立し、さらに行動条件付けでワールドモデルをロボット計画にまで接続しました。これは構想の「知覚 → ワールドモデル → アクター」という連携を、限定的ながら実機で実証した例です。第1段階で大量動画から知覚と物理常識を学び、第2段階で少量のロボット動画から行動の効き方を学ぶ分離は、構想が説く「知覚モジュールと世界モデルの分離」とよく対応します。

思想の対比として、生成型ワールドモデルの Dreamer 系(Hafner ら、DreamerV3 など)や、画像生成型の MAE(He ら, CVPR 2022)がよく引かれます。これらは観測の再構成を学習信号にする点で JEPA と異なり、ルカンが批判する「予測不能な細部を描く無駄」を抱えます。一方でこれらは「何を予測しているか」を映像として確認できる利点を持ち、優劣は単純ではありません。崩壊回避の系譜(BYOL, SimSiam, VICReg, Barlow Twins)も、JEPA の predictor + stop-gradient による非対称崩壊回避の土台として位置づけられます。

## 論文の実験結果(定量データ)
構想論文そのものには定量評価がありません。これは批判の対象にもなり、「壮大なビジョンだが検証可能な予測が乏しい」という指摘がなされてきました。そこで、構想の妥当性は後続論文の数値で間接的に測られます。

I-JEPA は、ImageNet(約128万枚・1000クラス)の線形プロービング(凍結特徴の上に1層分類器だけ学習する評価)で、ViT-Huge を用いて生成系 MAE を上回るトップ1精度(70%台後半)を、より少ない GPU 時間(MAE や iBOT のおよそ数分の1)で達成したと報告されました。これは「特徴空間予測は生成より効率的」という構想の主張を支持する数値です。

V-JEPA 系は、Something-Something v2(動きの理解を問う動画ベンチマーク。V-JEPA 2 でおよそ77%前後)など、見た目でなく時間方向の理解が必要なタスクで、ピクセル再構成系を上回ったと報告されています。V-JEPA 2-AC はさらに、行動ラベル・報酬なしの62時間規模の動画から学んだ世界モデルだけで、新物体・新環境のゼロショットに近い把持・配置をこなし、しかも1ステップの計画がおよそ16秒程度と生成型より桁違いに軽い、とされ、「観察から世界モデルを作り計画する」という構想の中核を限定的に実証しました。

ただし、H-JEPA(階層型)を end-to-end で大規模に動かし、長期計画まで通した定量的成功例はまだ報告されていません。階層をどう学習させるか、上位階層の抽象状態をどう自動で立ち上げるか、という最も野心的な部分は依然として未実証です。構想全体の検証は依然として部分的、というのが正確な現状評価です。

## メリット・トレードオフ・限界
メリットは、観察(ラベルなしの動画など)から効率的に世界モデルを学べる見込みがあること。予測型なので行動の結果を現実で試す前に頭の中でシミュレーションでき、安全性が高いこと。階層化により長期計画と短期反応を両立できる構造を持つことです。

トレードオフと限界は明確です。第一に、構想論文であり全体を end-to-end で実証したものはまだ存在せず、部分的実証にとどまること。第二に、特徴空間の予測は人間が正しさを直感的に検証しづらいこと。第三に、階層型・潜在変数の正則化・崩壊回避など、実装上の難しさが多く積み重なっていることです。とくに H-JEPA の階層をどう学習させるか、長期の抽象計画をどう具体行動に落とすか(階層間の橋渡し)は、現在も決定打がありません。また「世界モデルさえあれば常識が身につく」という前提自体への反論(記号的・論理的推論や言語の役割を過小評価している、という批判)もあり、構想は学界で賛否を呼んでいます。

研究上の未解決課題として、(1)階層的な抽象化を自動で学習する方法、(2)潜在変数による多峰的な未来表現と、その情報量の適切な制御、(3)エネルギーベース定式化の実用的な学習アルゴリズム(高エネルギー領域をどうサンプルするか)、(4)言語のような記号的計画と JEPA 的な連続的世界モデルの統合、が挙げられます。

## 発展トピック・研究の最前線
この構想は、JEPA 系([JEPA (Joint-Embedding Predictive Architecture)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0064_jepa-architecture), [I-JEPA (Image JEPA)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0059_i-jepa_note), [V-JEPA 2](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0101_v-jepa-2))の思想的背骨であり続けています。最前線は、V-JEPA 2-AC のように世界モデルを実際の制御へ接続する流れ([行動条件付きワールドモデル](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0003_action-conditioned-world-model))と、崩壊回避を理論的に理解する流れ([Stop-gradient (勾配停止)](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0658_stop-gradient), [VICReg](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0661_vicreg_note), [SimSiam](https://zenn.dev/riita10069/books/driving-automation-foundation/viewer/0093_simsiam))の二つです。自動運転の文脈では、H-JEPA の「短時間の細かい予測(直近の回避行動)と長時間の抽象計画(目的地までの経路)を階層で分ける」という発想が、知覚・予測・計画を分離する設計と相性が良く、特徴空間での未来予測を計画に結びつける研究の指針になっています。加えて、言語モデルが持つ記号的推論と、JEPA が持つ連続的・物理的な世界モデルを統合する方向(視覚言語行動モデルなど)が、構想の「言語と世界モデルの橋渡し」という未解決部分への現実的なアプローチとして注目されています。
